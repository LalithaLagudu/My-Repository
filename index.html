DevOps Lifecycle for Vizag Tourism Website

Plan

Define features & tasks (e.g., categories, filtering)

Develop

Write HTML, CSS, JS; use version control

Build

Bundle or minify code (optional for static sites)

Test

Check UI, responsiveness, functionality

Release

Tag version, prepare for deployment

Deploy

Host on GitHub Pages, Netlify, or similar

Operate

Monitor uptime & performance

Monitor

Track user behavior & errors
